# Pizza Bill Generator

## Overview

This project contains a java app with source files in `src/`.
Its a basic project to deepen java oop concepts. The project generates pizza bills for two pizza types "pizza" and "deluxe Pizza".

- There are Two types of Pizzas, 1. regular pizza and 2. Deluxe Pizza
- Each pizza type has both "veg" and "non-veg" options
- Extra Cheese and Toppings can be added in regular pizza but they are default in a deluxe pizza. Extra Charges are applied for both cheese and toppings seperately
- The order can be Take-away or Dine-in. Extra charges for back pack if opted for Take-away

## Requirements

- Java JDK 17 or newer
- Git
- VS Code (optional but recommended)
